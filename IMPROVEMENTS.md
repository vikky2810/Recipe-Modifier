# Improvement List
- [ ] Allow a patient to have multiple medical conditions at the same time.

- [ ] Add a field for custom instructions. If the user is missing ingredients, the recipe should adapt and regenerate without them.

- [ ] When generating a report, show only the current entry, not previous ones. Keep the report structured and clean.

- [ ] Calorie calculation for obesity should consider ingredient quantities, not just the ingredient list.
- [ ] Reduce response time
- [ ] Include the dish image at the top of the recipe instructions.
- [ ] Add recipe feedback and the option to regenerate the recipe.
